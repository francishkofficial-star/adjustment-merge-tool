SOH Report 库存合并工具

拖拽 SOH report Excel 文件，按预算模板 SKU 顺序合并库存

处理逻辑：
1. 读取 SOH report 的 C列（商品编码）、D列（商品名称）、S列（库存数量）
2. 按商品名称合并库存数量（求和），忽略 /箱出 后缀进行匹配
3. 对照 CSP商品管理模版 中的商品编码↔商品名称映射关系
4. 按 预算总表 的类目顺序输出，商品名称使用预算模板标准名称
5. CSP中有但SOH中没有的SKU，库存标0；SOH中有但CSP中没有的SKU，单独列出
📁

点击或拖拽 SOH report 文件到此处

支持 .xlsx / .xls 格式
